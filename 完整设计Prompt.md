## 使用建议

| 工具 | 用法 |
|------|------|
| **Midjourney** | 直接粘贴英文描述，加 `--ar 9:19 --v 6` |
| **DALL-E 3** | 直接粘贴中/英文描述 |
| **通义万相** | 粘贴中文描述效果更好 |
| **即梦** | 中英文混合使用 |
| **可灵** | 适合出图后进一步编辑 |

关键点：Midjourney 适合出概念图（风格参考用），通义万相/即梦对中文 UI 元素的识别更准。
