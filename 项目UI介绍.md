# 糖记 — 甜品饮品记录 App UI 介绍

## 项目概况

一款手机端甜品/饮品日记 App，PWA 架构，风格为**淡粉色调 + 圆润卡片 + 甜品可爱风**。全屏手机界面（390×844），所有页面圆角大、留白多、强调温暖治愈感。

## 页面清单

### 1. 首页（手账）

**布局：**
- 顶部导航栏：左侧「🍰 糖记」（点击跳日历），右侧「⚙️」设置
- 记录入口横幅：渐变粉色圆角条「📸 记录今天吃了什么」+ 右侧红底「＋」圆按钮
- 搜索框：🔍 图标 + 输入框 + 🤖 AI 搜索按钮（输入后出现）
- 时间线：按日期分组展示甜品卡片，日期行显示「X月X日 周X」+「N块🥄」
- 每张甜品卡片：图片（200px高渐变底色占位）+ 左上角标签 + 右上角删除/分享按钮 + 底部名称 + 店铺名 + 价格
- 底部导航栏：两个 Tab —「📖 手账」和「📊 统计」
- 右下角浮动 🎲 按钮（随机推荐命运骰子）

**视觉风格：** 卡片式纵向滚动，每张卡之间留白 16px，圆角 12px，轻微阴影

---

### 2. 记录流程（RecordFlow）

多步骤表单，底部进度指示器

**步骤 1 - 拍照/选图：**
- 顶部返回 + 标题「📸 记录」
- 中间图片区域（300px，点击拍照或从相册选）
- AI 识别中显示动画，识别成功后自动填入下方表单

**步骤 2 - 详情：**
- 名称输入框
- 分类切换：🍰 甜品 / 🧋 饮品（两个圆角按钮互斥）
- 甜度选择：无糖/微甜/适中/偏甜/很甜（横向标签选择器）
- 风味标签：抹茶/可可/咖啡/果味/花香/酒香/芝士/坚果/焦糖/椰香/茶味/豆乳（网格排列，可多选）
- 口感/配料标签（根据分类不同）
- 温度：常温/冷藏/热食/冰品

**步骤 3 - 补充信息：**
- 店铺名 + 位置
- 价格（数字输入）
- 评分（1-5，用🥄数量表示）
- 备注（多行文本）
- 日期选择（默认今天）
- 自制开关
- 提交按钮「✨ 保存记录」

---

### 3. 详情页

- 顶部导航：← 返回 + 右上角「⋯」编辑入口
- 300px 宽幅照片（或渐变粉色占位 + 甜品emoji）
- 评分区：大号 🥄 重复 + 数字评分 + 评语
- 标签行：甜度、口感、风味、温度（粉色圆角标签）
- 分类徽章：🍰 甜品 / 🧋 饮品（圆角）
- 名称：大字号居中
- 信息卡：📍店铺名（可点击进店铺详情）+ 位置 + ¥价格
- 备注卡（如有）：💬 备注内容
- 时间戳：底部居中
- 操作按钮行：📸 生成分享卡片 / ✏️ 编辑 / 🗑️ 删除（三个圆角按钮并列）

---

### 4. 统计页

- 顶部标题「统计」
- 四格数据卡片：本月·块 / 总计·块 / 本月花费 / 总计花费
- 本周总结卡片：周日期范围 + 块数/评分/花费三列 + 每日柱状图 + 本周最受欢迎风味
- 右上角「完整周报 →」入口
- 月度趋势：6个月柱状图
- 店铺排行榜：🥇🥈🥉 排名 + 店名/次数/评分 + 进度条
- 评分分布：5️⃣4️⃣3️⃣2️⃣1️⃣ 水平条形图
- 分类统计：🍰甜品 vs 🧋饮品 占比
- 口味偏好：风味名称 + 百分比进度条
- AI 口味报告卡片：生成按钮 + AI 生成的分析文案
- 「📋 想吃清单 →」底部跳转

---

### 5. 周报页

- 顶部导航：← 返回 + 📊 本周周报 + 日期范围
- AI 周报卡片：渐变棕红色背景，白色文字，AI 自动生成的每周总结
- 核心数据三列：本周·块 / 平均评分 / 本周花费
- 本周最佳：卡片展示（缩略图 + 名称 + 评分 + 店铺）
- 每日分布：7天柱状图（周一到周日）
- 风味排行：TOP 5 横向条形图，带序号和颜色
- 常去店铺：🥇🥈🥉 排名列表
- 分类 + 评分分布：两列布局，半宽卡片
- 底部「🔄 重新生成 AI 分析」按钮

---

### 6. 店铺详情页

- 顶部导航：← 返回 + 📍店名 + 按日期/按评分切换
- 店铺英雄区：渐变棕色背景 + 店名 + 去过次数/平均评分/总计花费三列 + 最佳记录
- 风味摘要行：🍭 常见风味列表
- 记录列表：按日期分组或按评分排序，同首页卡片样式

---

### 7. 日历页

- 顶部导航：🍰 糖记（返回首页）
- 月视图日历网格
- 有记录的日期有标记
- 点击日期弹出当日摘要：吃了N块 + 平均评分 + 小卡片预览

---

### 8. 想吃清单

- 顶部导航：← 返回 + 📋 想吃清单
- 添加输入框 + 按钮
- 列表项：内容 + ✕ 删除

---

### 9. 设置页

- 顶部导航：⚙️ 设置
- 导出记录（JSON 文件）
- 导入记录（JSON 文件选择）
- 记录数量统计
- 清除所有数据（危险操作）

## 整体交互风格

- **圆角规范**：卡片 12px、按钮 24px（pill）、弹窗 16px
- **阴影**：细腻的粉色系阴影 `rgba(232, 113, 109, 0.08)`
- **过渡动画**：卡片有微缩放开启动效（scale 0.99）
- **滚动**：隐藏滚动条，模拟原生 App 体验
- **加载状态**：AI 操作时有脉冲动画 + 状态文字（如「🤖 AI 正在分析…」）
- **空状态**：大号 emoji + 友好文案（「🍨 还没有记录 / 你的第一条记录正在等你」）
- **弹窗**：半透明黑色遮罩 + 白色居中圆角卡片
