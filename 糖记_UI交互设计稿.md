# 糖记 — 完整 UI 交互设计稿

---

## 全局设计 Token

| 维度 | 取值 |
|------|------|
| 主色调 | 奶油白 `#FFFBF5` 底，焦糖棕 `#8B5E3C` 主文字 |
| 强调色 | 草莓红 `#E8716D`（勺子、按钮、成就徽章） |
| 辅助色 | 抹茶绿 `#7DA87B`（成功/已完成）、奶油黄 `#F5D6A8`（标签底色） |
| 字体 | 中文：系统默认（苹方/思源），英文/数字：Inter 或 SF Rounded |
| 圆角 | 全局 16px，卡片 20px，按钮 24px（胶囊形） |
| 间距 | 基础单位 4px，全局 padding 20px |
| 阴影 | 极轻，`0px 2px 12px rgba(139, 94, 60, 0.08)` |
| 动效 | 默认 300ms `ease-out`，弹簧动画 `spring(0.7, 0.9)` |

### 交互
- **下拉**：显示日历月视图（见屏幕 1A）
- **点击卡片**：进入详情（见屏幕 4）

#### 勺子动画状态机

| 分数 | 盘子状态 | 勺子位置 | 文案 |
|------|---------|---------|------|
| 0.5 | 几乎完整 | 刚碰到 | 就尝了一小口 |
| 1.5 | 缺一角 | 挖了一勺 | 还可以 |
| 2.5 | 缺了 1/4 | 正在挖第二勺 | 还行吧 |
| 3.5 | 缺了 1/3 | 第三勺悬停 | 好吃的！ |
| 4.5 | 缺了一半 | 正在挖第四勺 | 非常好吃！ |
| 5.0 | 快吃完了 | 最后一勺 | 此物只应天上有 |

抖动反馈：勺子划过整数/半整数时，盘子微微弹跳，手机短震。

## 动效清单

| 场景 | 动画 | 时长 |
|------|------|------|
| 首页卡片出现 | 从下方 20px 淡入 + 上移 | 400ms |
| 勺子评分拖动 | 实时跟手，松手弹簧回弹 | 实时+300ms |
| 完成记录 | 卡片弹射飞入 + 计数器跳变 | 500ms |
| 成就弹出 | 底部弹入 + 徽章旋转出现 | 600ms |
| Tab 切换 | 交叉淡入淡出 | 250ms |
| 日历展开/收起 | 高度展开弹性动画 | 350ms |
| 照片全屏 | 从卡片位置放大到全屏 | 300ms |
| 删除记录 | 卡片缩小 + 淡出 | 250ms |
| 标签选中 | 胶囊颜色过渡 + 弹性缩放 | 200ms |
| 页面转场 | 右滑进入，左滑返回（iOS 标准） | 300ms |
| 下拉刷新 | 勺子旋转 360° | 持续到释放 |

## 数据模型（极简）

```
dessert_record
├── id (UUID)
├── image_path (本地路径)
├── rating (Float, 0.5~5.0, 步长 0.5)
├── sweetness (Enum: 微甜/适中/浓郁/齁甜)
├── texture (多选: 绵密/轻盈/酥脆/Q弹/流心/冰沙…)
├── flavor (多选: 抹茶/可可/咖啡/果味/花香/酒香…)
├── temperature (Enum: 常温/冷藏/热食/冰品)
├── shop_name (String, 可选)
├── location (String, 可选, 手动输入地址)
├── price (Float, 可选)
├── note (String, 可选)
├── is_homemade (Bool)
├── recipe_note (String, 可选)
├── created_at
└── updated_at
```

全部本地，一表到底，不需要关系型外键。
